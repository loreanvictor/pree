pree is a command line utility that prebuilds HTML files, and:

 🧬 pre-renders web components(faster loading, MUCH smoother UX) 🏗️ handles layouting and metadata (no copy-pasta, breezy DX) 👻 manages build-only components and scripts (even faster loading) ✨ gets components and scripts access to build environment (for some magical web components) 
pree view     <src>                      # 👉 preview your website
pree build    <src> <dest>               # 👉 prebuild your website
