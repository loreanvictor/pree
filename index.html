pree is a command line utility that prebuilds HTML files, and:

 🧬 pre-renders web components (instant page load) 🏗️ handles layouting and metadata (smooth DX) 👻 manages build-only components and scripts (small bundles) ✨ gets components and scripts access to build environment (for some magical web components) 
pree view     <src>                      # 👉 preview your website
pree build    <src> <dest>               # 👉 prebuild your website
