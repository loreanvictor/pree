--- title: "pree: installation" layout: ./_layout.html --- ## Installation _You need [Node.js](https://nodejs.org/en/) to run `pree`._ ⬇️ Install `pree`: ```bash npm i -g pree ``` 📦 Check for updates: ```bash pree version # tells you if you need to update ``` ```bash npm i -g pree@latest ``` --- 🚀 You can also use [npx](https://www.npmjs.com/package/npx) to run `pree` without installing it: ```bash npx pree view ```
