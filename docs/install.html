--- title: "pree: installation" layout: ./_layout.html --- ## Installation You need [Node](https://nodejs.org/en/). Install `pree` globally, or use [npx](https://www.npmjs.com/package/npx): ```bash npm i -g pree # ⬇️ installs the CLI ``` ```bash npx pree view # 🚀 runs the CLI without installing it ```
To update (or to ensure you are using the latest version), use `@latest` tag: ```bash npm i -g pree@latest ``` ```bash npx pree@latest view ```
👉 Check for updates: ```bash pree version ```
