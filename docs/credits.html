Puppeteer Node.js API for Chrome. Used to prebuild HTML files and pre-render web components using [declarative shadow DOM](https://developer.chrome.com/docs/css-ui/declarative-shadow-dom).
Koa Next generation web framework for Node.js. Used for serving files either in `pree view` or in `pree build`. Koa middlewares also handle [metadata and layouts](usage/meta-layout)
Cheerio The fast, flexible & elegant library for parsing and manipulating HTML and XML. Used for manipulating HTML files for [layouting](usage/meta-layout), [removing build-only scripts](usage/build-only-scripts), injecting code, etc.
minimatch A glob matcher in javascript. Used for figuring out which files to build.
Chalk Terminal string styling done right. Used to do right terminal styling.
Yargs Yargs be a node.js library fer hearties tryin' ter parse optstrings.
YAML YAML parser and stringifier for JavaScript. Used for parsing [config](usage/config).
gray-matter Smarter YAML front matter parser. Used for handling [metadata](usage/meta-layout)
Simple Git A light weight interface for running git commands in any node.js application. Used for providing git related [build environment APIs](components/build-env).
Chokidar Minimal and efficient cross-platform file watching library. Used for live-reloading.
ws Simple to use, blazing fast and thoroughly tested WebSocket client and server for Node.js. Used for live-reloading.
libnpmsearch Node.js library for programmatically accessing the npm search endpoint. Used to determine the latest published version of `pree` and if an update is required.
semver The semver parser for node (the one npm uses). Used to check whether an update for `pree` is required.
http-errors Create HTTP Errors. Used for for error handling of [build environment APIs](components/build-env).
Mime Mime types for JavaScript. Used for determining file types, proper response types, and detecting HTML files that need to be layouted / manipulated, etc.
find-package-json Easily find package.json files. Used to determine locally installed version of `pree` and if an update is required.
git url parse A high level git url parser for common git providers. Used for providing git related [build environment APIs](components/build-env).
filesize.js JavaScript library to generate a human-readable string describing the file size. Used to display file sizes in directory view.
ms Tiny millisecond conversion utility. Used to display how long CLI tasks take.
