`pree` is a minimalistic tool that prebuilds HTML files, and: 🧬   it pre-renders web components (faster loading, MUCH smoother UX) \ 🏗️   it handles layouting and shared metadata (no copy-pasta, breezy DX) \ 👻   it enables build-time only components and scripts (even faster loading) \ ✨   it provides build environment access to components and scripts (for some magical web components)
```bash pree view # 👉 preview your website pree build # 👉 prebuild your website ```
