`pree` is a command line utility that prebuilds HTML files, and: 🧬 pre-renders web components (instant page load) 🏗️ handles layouting and metadata (smooth DX) 👻 manages build-only components and scripts (small bundles) ✨ gets components and scripts access to build environment (for some magical web components) ```bash pree view # 👉 preview your website pree build # 👉 prebuild your website ```
