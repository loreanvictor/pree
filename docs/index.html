A [CLI](https://en.wikipedia.org/wiki/Command-line_interface) that prebuilds HTML files. Use it to build websites with plain HTML, CSS and JS. 🧬 It pre-renders web components (instant page load) 🏗️ It handles layouting and metadata (smooth DX) 👻 It manages build-only components and scripts (small bundles) ✨ It gets components and scripts access to build environment (for some magical web components) ```bash pree view # 👉 preview your website pree build # 👉 prebuild your website ```
